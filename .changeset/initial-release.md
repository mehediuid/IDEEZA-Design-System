Initial release of the IDEEZA Design System.

- `@ideeza/tokens` ships all 11 token collections from the Figma source: 68 primitives, 120 semantic colors with light/dark, responsive spacing/radius/typography, motion, elevation, opacity, border width, z-index, and safe areas. Distributed as CSS variables, TS constants, and a Tailwind preset.
- `@ideeza/ui` ships the `Button` component with five variants (primary, secondary, blue, ghost, danger) and four sizes, supporting `asChild`, icons, and a loading state.
